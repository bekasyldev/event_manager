# EventManager

Приложение для управления мероприятиями на Next.js 16, React 19, TypeScript.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

Остальные команды: `npm run build`, `npm run start`, `npm run lint`.

## Архитектурные решения

**Состояние** — Zustand (`store/useEventStore.ts`) с middleware `persist` (данные сохраняются в `localStorage`). Фильтрация, сортировка и пагинация реализованы внутри стора через вычисляемые методы (`getFilteredEvents`, `getPaginatedEvents`, `getTotalPages`).

**Фильтры в URL** — через библиотеку nuqs (`hooks/useFilters.ts`). При изменении фильтров URL обновляется, поэтому состояние фильтров сохраняется при перезагрузке и доступно по ссылке. Поисковый запрос дополнительно дебаунсится на 300 мс (`hooks/useDebounce.ts`).

**Формы** — React Hook Form + Zod (`lib/schemas.ts`). Валидация только при сабмите, без лишних ре-рендеров.

**Компоненты** — UI-слой отделён от логики. Страницы (`app/`) подключают стор и передают обработчики вниз. Все переиспользуемые утилиты вынесены в `lib/`.

## Реализованные опциональные функции

- **Фильтры в URL** — поиск, категория, статус и сортировка синхронизированы с адресной строкой
- **Страница избранного** — отдельный маршрут `/favorites`
- **Экспорт в JSON** — кнопка выгружает текущий отфильтрованный список мероприятий
- **Пагинация** — по 6 событий на странице, скрывается если страница одна
- **Валидация даты** — форма не принимает прошедшие даты
